COLORS AND ANIMATIONS

Color palette, dark theme and animations

This assignment consisted of three parts. Part one was to create a color palette, using the 60/30/10 method. We sould also validate if the contrast of the colors in a contrast checker. In part two our task was to create a dark theme using CSS varaibles and local storage. The design should shift between dark and light theme with a toggle button. Part three was to add animations to our site, preferabel following Disneys 12 rules for animation that we learned about in class.
